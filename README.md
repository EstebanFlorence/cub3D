## Usage
cub3D uses the [miniLibX](https://github.com/42Paris/minilibx-linux) library

**Clone the Repository**
  ```shell
  git clone git@github.com:EstebanFlorence/cub3D.git
  cd cub3D
  ```

 **Compile on Linux**
  ```shell
  make
  ```
 **Compile on MacOS**
  ```shell
  make mac
  ```
  **Run**
  ```
  ./cub3D <file>.cub
  ```

NB  \<file\>.cub must be located in maps/
